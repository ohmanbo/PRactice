# Pull Request(PR)을 실습하기 위한 Repository(이하 Repo)입니다.
---

1. 본 Repo를 본인의 GitHub에 Fork합니다.
2. 본인의 로컬에 Clone합니다.
3. 본인 이름(국문)의 폴더명을 본인의 이름(영어)으로 변경합니다.
   - Hint: mv
4. 자신의 폴더 안에 다음 내용을 반영하여 README파일을 수정합니다.
   - README파일의 코더를 본인의 이름으로, 리뷰어를 퍼실의 이름(아무나)으로 변경합니다.
   - 1, 3, 5번 체크박스에 체크합니다.
      - markdown에서 체크는 X입니다. ([ ] - [x])
   - 체크를 원하지 않는 나머지 번호는 비워둡니다.
   - 1번 항목에 이미지를 추가합니다.
      - 본인이 활용한 CLI의 화면을 캡쳐하여 추가합니다.
   - 변경사항을 반영하여 push합니다.
5. Pull-Request를 보냅니다.
